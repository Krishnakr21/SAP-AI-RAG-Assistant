# AI-Powered SAP-Integrated Document Retrieval System (RAG + Azure OpenAI)

## Overview
This project implements an AI-powered document retrieval system integrated with SAP FI/CO datasets. It leverages a **RAG (Retrieval-Augmented Generation) pipeline** along with **Azure OpenAI** to enable semantic search and automated insights from business reports and exports.

## Features
- **Optimized RAG Pipeline:** Enables semantic search across SAP FI/CO datasets for fast and accurate retrieval of financial information.
- **Flask-Based Backend:** Provides low-latency responses for financial insights using embedding retrieval.
- **Automated Agentic Workflows:** Reduces manual effort in SAP data analysis and enhances reporting accuracy.

## Technologies Used
- Python
- Flask
- Azure OpenAI
- SAP FI/CO data integration
- Semantic search & embeddings

## Project Highlights
- Built an efficient retrieval system for SAP datasets.
- Developed AI-powered automation workflows for financial reporting.
- Leveraged ChatGPT for project design and implementation guidance.
